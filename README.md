
# Multisig Transaction History

This script allows to pull the entire transaction history from a Multisig wallet (Gnosis Multisig or GSNMultisig)

## Getting Started

### Install dependencies

```
$ npm install
```

### Configure environment variables

```
$ mv .env_sample .env
$ vi .env
// .env
INFURA_KEY=
MNEMONIC=
```

### Run the script

```
node index.js
```
